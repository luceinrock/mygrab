# Plan 01-06 — Driver App: Profile Editing Screen

## Objective
Drivers have no way to update their vehicle info or name from the app.
Build a Profile screen accessible from the Home screen.

## Context
- @android-driver/app/src/main/java/com/ridenow/driver/ui/home/HomeScreen.kt
- @android-driver/app/src/main/java/com/ridenow/driver/ui/home/HomeViewModel.kt
- @android-driver/app/src/main/java/com/ridenow/driver/data/remote/RideNowApi.kt
- @android-driver/app/src/main/java/com/ridenow/driver/ui/navigation/NavGraph.kt
- Backend: PUT /api/v1/drivers/profile — accepts full_name, vehicle_make, vehicle_model, vehicle_color, plate_number, year_manufactured

## Tasks

### Task 1 — Create ProfileViewModel + ProfileScreen
**Layer:** Android driver app
**Builds on:** RideNowApi already has getProfile() and updateProfile() (PUT exists via Retrofit); Hilt DI already wired
**Action:**
1. Create `ui/profile/ProfileViewModel.kt`:
   - Load current profile on init via getProfile()
   - `save(fullName, vehicleMake, vehicleModel, vehicleColor, plateNumber)` calls PUT /drivers/profile
   - UiState: loading, success, error
2. Create `ui/profile/ProfileScreen.kt`:
   - Pre-filled form: Full Name, Vehicle Make, Model, Color, Plate Number
   - Save button; shows success snackbar or error message
   - Back navigation
**Done:** ProfileScreen compiles; driver can edit and save their details; changes persist (verified by re-opening screen).

### Task 2 — Wire profile navigation
**Layer:** Android driver app (NavGraph.kt, HomeScreen.kt)
**Builds on:** NavGraph already has auth + home + ride routes
**Action:**
1. Add "Profile" route to NavGraph.kt
2. Add a profile icon button to HomeScreen top bar that navigates to Profile
**Done:** Tapping profile icon opens ProfileScreen; back returns to Home.

## Verification
Build the driver app module:
```bash
cd android-driver && ./gradlew :app:compileDebugKotlin
```
No compilation errors.

## Checkpoint
`checkpoint:human-verify` — Install APK on device; edit vehicle color; reopen profile; confirm change saved.
