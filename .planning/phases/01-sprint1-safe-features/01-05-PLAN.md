# Plan 01-05 — Admin: Driver Location Map View

## Objective
The online drivers panel shows lat/lng as a text link. Give admins a proper map
that shows all online drivers as markers, updating in real-time.

## Context
- @admin-web/src/pages/Dashboard.tsx — online drivers list with lat/lng
- @admin-web/src/pages/Drivers.tsx — Online tab with driver cards
- @admin-web/src/lib/supabase.ts — Supabase client (Realtime already wired)
- Online drivers endpoint: GET /api/v1/admin/drivers/online

## Tasks

### Task 1 — Add a map view to the Drivers Online tab
**Layer:** Admin web (Drivers.tsx)
**Builds on:** Online tab already fetches drivers with lat/lng; Realtime subscription already set up
**Action:**
1. Install `npm install react-map-gl maplibre-gl` in admin-web
2. Add a "Map / List" toggle button on the Online tab header
3. Map view: render a MapLibre GL map (free tiles, no API key needed) centered on Cebu
   (10.3103, 123.8849), zoom 12
4. For each online driver with lat/lng: render a green marker; on hover/click show a
   popup with name, vehicle, plate, last-seen age
5. Realtime subscription already triggers re-fetch — map updates automatically
**Done:** Switching to Map view shows driver markers; markers update within seconds of
a driver going online/offline. No API key required.
