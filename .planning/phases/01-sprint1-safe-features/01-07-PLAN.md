# Plan 01-07 — Driver App: In-App Navigation Shortcut

## Objective
When a driver accepts a ride, they need to navigate to pickup then dropoff.
Add "Navigate" buttons that open Google Maps (or Waze) with the destination pre-filled.

## Context
- @android-driver/app/src/main/java/com/ridenow/driver/ui/ride/ActiveRideScreen.kt
- @android-driver/app/src/main/java/com/ridenow/driver/ui/ride/RideViewModel.kt
- Ride object has: pickup_lat, pickup_lng, dropoff_lat, dropoff_lng, pickup_address, dropoff_address

## Tasks

### Task 1 — Add navigation launcher utility
**Layer:** Android driver app
**Builds on:** Ride data already available in RideViewModel uiState
**Action:**
1. Create `utils/NavigationUtils.kt`:
   ```kotlin
   fun openNavigation(context: Context, lat: Double, lng: Double, label: String) {
       val uri = Uri.parse("google.navigation:q=$lat,$lng&mode=d")
       val intent = Intent(Intent.ACTION_VIEW, uri).apply {
           setPackage("com.google.android.apps.maps")
       }
       // Fallback: geo URI if Maps not installed
       if (intent.resolveActivity(context.packageManager) != null) {
           context.startActivity(intent)
       } else {
           context.startActivity(Intent(Intent.ACTION_VIEW, Uri.parse("geo:$lat,$lng?q=$lat,$lng($label)")))
       }
   }
   ```
2. In ActiveRideScreen.kt:
   - When status is `accepted` or `arrived`: show "Navigate to Pickup" button
   - When status is `in_progress`: show "Navigate to Dropoff" button
   - Each button calls openNavigation() with the appropriate lat/lng
**Done:** Tapping "Navigate to Pickup" opens Google Maps navigation to pickup coords.
